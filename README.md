# FDF

## Project Overview

- The representation in 3D of a landscape is a critical aspect of modern mapping.
- The program takes a map as parameter and creates its 3d representation. 

## How to compile:
- git clone https://github.com/alexSolohin/FDF.git
- cd FDF
- make

## How to run
- ./fdf test_maps/maps
